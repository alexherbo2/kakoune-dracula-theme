### [Kakoune]

#### Install

Add [`dracula.kak`] and [`dracula-transparent.kak`] to your [`colors`] directory.

#### Activating theme

``` kak
colorscheme dracula
```

[`colors`]: https://github.com/mawww/kakoune#color-schemes
